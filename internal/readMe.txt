Инструкция:
- Нужно установить postgres
- Миграции не нужны, сервис сам создаст таблицы.

1. Запускаем Сервер (cmd/server/main.go) с флагом -d="dns для подключения к posgres"
2. Запускаем Клиент (cmd/client/main.go) без флагов или готовый клиент из папки /build
3. Дальше в меню:
 - NewUser - Login, Register
 - Store Data - создать и записать данные
 - Load Data  - загрузить данные из базы

FTP Server для скачки клиентов по адресу: 127.0.0.1:8082/static
